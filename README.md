This deck uses the WaniKani method and data for the kanji and kanji-related vocabulary from the Genki I 3rd edition book.

The vocabulary decks use sound files from WaniKani.

Each Genki lesson has 5 decks:
<ul>
    <li>0 Radicals: Radicals are the kanji components and should be learned first. (Or ignored if you don't like learning radicals)</li>
    <li>1 Kanjis: These are the kanjis in the order Genki wants you to learn them.</li>
    <li>2 Vocabulary: This deck contains all the vocabulary from Genki that is listed with the kanji and Genki marks as important and some more I deemed important.</li>
    <li>3 Unimportant Vocabulary: This deck contains all the vocabulary that Genki does not mark as important. Removed words with kanji that appear in later lessons. (May need more pruning.)</li>
    <li>4 Additional Vocabulary: This deck contains the WaniKani vocabulary that uses kanjis from this and previous lessons, and is not mentioned by Genki.</li>
</ul>

I recommend learning 0, 1 and 2 (and 3 if you have time).

The WaniKani level decks contain the radicals, kanji, vocabulary, and kana vocabulary not already covered in the Genki lesson decks.

This deck was generated via Python. Contributions welcome.
